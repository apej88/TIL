# 튜플 (Tuple)
- 리스트와 유사한 집합적 자료형이지만
- 리스트와 달리 원소 변경 불가
    - 추가 / 수정 / 삭제 다 안 됨
- 소괄호 () 사용
- 튜플 = (값1, 값2, … , 값n)
- colors = ("red", "green", "blue")
- numbers = (1, 2, 3, 4, 5 )
- 각 원소는 인덱스(Index)로 구분(인덱스: 0부터 시작) 
- 원소 변경 시 에러
- numbers[0] = 10 #오류
- 튜플을 사용하는 이유
    - 실수로 요소가 추가되거나 삭제되지 않도록 
    - 내용을 변경할 수 없게 하기 위해서 사용

## 튜플 만들기
- t1 = (1, 2, 3)
- t2 = 4, 5, 6  # 괄호 없어도 튜플로 생성됨
- t3 = t1, (7, 8, 9) # 튜플 내에 다른 튜플 포함
- t4 = [1, 2], [3, 4] # 리스트로 튜플 생성
- t5 = tuple([5, 6, 7, 8]) # tuple() 함수 사용
    - 리스트를 튜플로 변환
